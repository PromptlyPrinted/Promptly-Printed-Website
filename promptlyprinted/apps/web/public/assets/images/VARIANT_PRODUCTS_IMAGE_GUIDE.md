## Notes

1. All images should be named exactly `image.png`
2. Use PNG format for transparency support
3. Square images (1:1 aspect ratio) work best
4. Minimum recommended size: 800x800px
5. Maximum recommended size: 2000x2000px
6. Images will be displayed on the `/products` page
7. The database has been updated with these image paths
8. Variant selection (size, finish, etc.) happens on the product detail page
