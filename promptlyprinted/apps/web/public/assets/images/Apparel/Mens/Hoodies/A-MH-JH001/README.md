# Men's Premium Organic Cotton Hoodie (A-MH-JH001)

This folder contains all images for the Men's Premium Organic Cotton Hoodie.

## Folder Structure

### `/Blanks/png/`
Place PNG images of the blank hoodie in different colors here.
Required color images (based on colorOptions in products.ts):
- `black.png` - Black hoodie
- `white.png` - White hoodie
- `navy.png` - Navy hoodie
- `heather-grey.png` - Heather Grey hoodie

These images should show the blank hoodie (no design) from the front view.

### `/Blanks/jpeg/`
Place JPEG versions of the same blank hoodie images here (optional, for performance).

### `/People/`
Place lifestyle images showing people wearing the hoodie here.
Examples:
- Person wearing the hoodie in various settings
- Multiple angles/poses
- Different demographics

### `/ProductImage/`
Place the main product thumbnail image here:
- `image.png` - Main product image used as the default thumbnail on product listing pages

This should be the most representative image of the product (typically white or black color).

## Image Requirements
- **Format**: PNG for blanks (transparent background preferred), JPEG for people photos
- **Size**: Minimum 1000x1000px for product images
- **Background**: White or transparent for blank products
- **Quality**: High resolution, professional quality
- **Consistency**: All colors should have the same angle, lighting, and composition
